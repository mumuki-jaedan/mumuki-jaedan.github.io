## 1. 변경 요약

### 1.1 프로젝트 구조 리팩토링

Quarto 소스 파일과 빌드 출력 디렉토리를 분리하여, 배포 대상(`docs/`)에 빌드 결과만 포함되도록 변경했다.

**변경 전:**
```
mumuki/
├── docs/
│   ├── _quarto.yml      ← 소스
│   ├── index.qmd        ← 소스
│   ├── custom.scss       ← 소스
│   ├── about/*.qmd      ← 소스
│   └── _site/           ← 빌드 출력
```

**변경 후:**
```
mumuki/
├── _quarto.yml           ← 소스 (루트)
├── index.qmd             ← 소스 (루트)
├── custom.scss            ← 소스 (루트)
├── about/*.qmd           ← 소스 (루트)
├── programs/*.qmd
├── transparency/*.qmd
├── participate/*.qmd
├── news/*.qmd
├── legal/*.qmd
├── assets/               ← 브랜드 에셋
├── images/               ← SVG 일러스트
└── docs/                 ← 빌드 출력만 (배포 대상)
    ├── index.html
    ├── about/
    ├── programs/
    ├── ...
    ├── assets/           ← Quarto가 복사
    ├── images/           ← Quarto가 복사
    └── site_libs/
```

**변경 이유:**

- `output-dir: _site`일 때 배포 대상이 `docs/_site/`가 되어 GitHub Pages/Cloudflare Pages 설정과 불일치
- `output-dir: .` 설정은 Quarto가 "Source and destination cannot be the same" 에러 발생
- 소스를 루트로, 출력을 `docs/`로 분리하는 것이 Quarto의 표준 패턴

### 1.2 _quarto.yml 변경사항

```yaml
project:
  type: website
  output-dir: docs          # _site → docs
  render:                   # 비웹사이트 파일 제외
    - "*.qmd"
    - "about/"
    - "programs/"
    - "transparency/"
    - "participate/"
    - "news/"
    - "legal/"
```

- `render:` 필드 추가로 CLAUDE.md, PLAN.md, PROGRESS.md 등이 빌드에 포함되지 않도록 제한
- Google Fonts `include-in-header` 추가: Noto Sans KR, Noto Serif KR 웹폰트 로드

## 2. 디자인 개선 상세

### 2.1 index.qmd 리뷰 이슈 및 수정

| 이슈 | 원인 | 수정 |
|------|------|------|
| `:::` 마커 텍스트 노출 | 6단계 중첩 fenced div + `page-layout: full` 호환 문제 | `page-layout: custom` 전환, 단순 fenced div 구조로 재작성 |
| Quarto 기본 타이틀 블록 중복 | `title:` 설정 시 자동 생성 | `title-block-style: none` + CSS `.quarto-title-block { display: none }` |
| section 하단 gold bar 중복 | Quarto가 `.section-heading`을 h2와 부모 section 양쪽에 적용 | `section.section-heading::after { display: none }` |

### 2.2 custom.scss 개선 (Tufte 스타일 강화)

**Hero 섹션:**
- Flexbox 2단 레이아웃 (콘텐츠 + 일러스트)
- 타이틀: Noto Serif KR 3rem, 딥틸 악센트 색상
- 서브타이틀: 48ch 최대 폭, 줄간격 1.8
- CTA 버튼: box-shadow + hover 애니메이션 추가

**프로그램 카드:**
- CSS Grid 3열 레이아웃 (`grid-template-columns: repeat(3, 1fr)`)
- SVG 아이콘 추가 (icon-book, icon-building, icon-heart)
- hover 시 border-color 변경 + translateY(-3px) 효과
- 좌측 정렬로 Tufte 가독성 원칙 준수

**통계 섹션:**
- 3열 stats-grid, 각 항목 white 카드 배경
- transparency-note: gold 좌측 보더 + 반투명 배경

**CTA 섹션:**
- Deep Teal (#2C5F5C) 전면 배경
- 흰색 텍스트 + 코랄/아웃라인 버튼 대비

**반응형:**
- 992px 이하: hero flex-direction column, 카드 1열
- 576px 이하: 버튼 block 전환, 타이틀 축소

### 2.3 Google Fonts 로드

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@300;400;500;600;700&family=Noto+Serif+KR:wght@400;700&display=swap" rel="stylesheet">
```

- `preconnect`로 DNS/TLS 핸드셰이크 사전 수행
- `display=swap`으로 FOIT(Flash of Invisible Text) 방지
