## 1. 수행 내역

### 구조 설계

```
docs/
├── _quarto.yml           # 프로젝트 설정 (navbar, footer, search, SEO)
├── custom.scss            # Tufte 스타일 SCSS (12개 컴포넌트 클래스)
├── index.qmd              # 홈페이지 (히어로, 프로그램 카드, 투명성 통계)
├── about/                 # 재단소개 (3페이지)
├── programs/              # 사업안내 (3페이지)
├── transparency/          # 투명한 운영 (3페이지)
├── participate/           # 참여하기 (3페이지)
├── news/                  # 소식 (listing + 1 포스트)
├── legal/                 # 법적 페이지 (2페이지)
├── images -> ../../images # 심링크 (SVG 일러스트레이션)
└── assets -> ../../assets # 심링크 (로고, 브랜드)
```

### 빌드 결과

- **총 17개 HTML 페이지** 렌더링 성공
- **사이트 크기**: 1.7MB
- **빌드 도구**: Quarto 1.9.16
- **에러**: 0건
- **경고**: div 중첩 경고 (Quarto 내부 템플릿, 기능 무영향)

## 2. _quarto.yml 핵심 설정

| 설정 | 값 | 근거 |
|------|------|------|
| `project.type` | website | 정적 사이트 |
| `format.html.theme` | cosmo + custom.scss | Tufte 스타일 커스터마이징 |
| `format.html.lang` | ko | 한국어 UI 전체 적용 |
| `website.search` | overlay | 내장 전문 검색 |
| `website.open-graph` | true | SNS 공유 최적화 |
| `website.favicon` | mumuki-favicon.svg | 브랜드 아이콘 |

## 3. custom.scss 컴포넌트

| 클래스 | 용도 |
|--------|------|
| `.hero-section`, `.hero-title`, `.hero-subtitle` | 홈페이지 히어로 영역 |
| `.hero-title-accent` | 제목 강조색 (인라인 스타일 제거) |
| `.btn-donate`, `.btn-outline` | CTA 버튼 (Coral, Teal) |
| `.program-card` | 프로그램 카드 (hover 효과) |
| `.feature-grid` | 자동 맞춤 그리드 (auto-fit, 260px) |
| `.sidenote` | Tufte 스타일 사이드노트 |
| `.stat-number`, `.stat-number-sm` | 통계 숫자 강조 |
| `.stat-label` | 통계 라벨 |
| `.key-value` | 키-값 표시 |
| `.illustration` | SVG 일러스트 컨테이너 |
